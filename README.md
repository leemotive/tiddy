# Tiddy

基于 Element Plus 的 Vue 3 组件库，提供增强的表单和表格实现。

## ✨ 特性

- **表单组件**：支持使用 JSON 配置表单
- **表格组件**：功能增强的数据表格
- **对话框组件**：灵活的对话框管理
- **TypeScript 支持**：完整的 TypeScript 类型定义
- **组合式 API**：基于 Vue 3 组合式 API 构建

## 🚀 安装

```bash
# 使用 pnpm (推荐)
pnpm add tiddy 

# 使用 npm
npm install tiddy 

# 使用 yarn
yarn add tiddy 
```
